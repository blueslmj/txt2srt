⚠️ 重要：系统要求
═══════════════════════════════════════
    
    需要 Python 3.10-3.13 版本
    推荐：Python 3.12 或 3.13
    
    ❌ Python 3.14 暂不支持
    如遇到版本问题，请查看：INSTALL_FIX.md

第1步：安装环境
═══════════════════════════════════════

    双击运行：setup.bat
    
    ⏱️ 预计时间：3-5分钟
    ✅ 会自动安装所有依赖


第2步：启动UI界面
═══════════════════════════════════════

    方式A（推荐）- 美观的Web界面：
    👉 双击运行：start_ui.bat
    
    方式B - 传统桌面界面：
    👉 双击运行：start_tkinter_ui.bat


第3步：使用工具
═══════════════════════════════════════

    1. 上传音频文件（MP3, WAV, FLAC等）
    2. 输入或上传文本内容
    3. 点击"开始处理"
    4. 下载生成的SRT字幕文件
    
    就这么简单！✨


📖 详细文档
═══════════════════════════════════════

    - QUICKSTART.md  → 快速开始指南（推荐阅读）
    - UI_GUIDE.md    → UI界面详细使用说明
    - README.md      → 完整功能文档
    - example.py     → Python代码示例


❓ 遇到问题？
═══════════════════════════════════════

    1. 查看 QUICKSTART.md 中的"常见问题"部分
    2. 确保已正确运行 setup.bat
    3. 检查是否使用了虚拟环境(venv)


🎯 推荐使用流程
═══════════════════════════════════════

    新手 → 阅读 QUICKSTART.md → 使用 start_ui.bat
    
    进阶 → 阅读 example.py → 使用Python API批量处理
    
    专家 → 直接使用命令行 txt2srt.py


💡 提示
═══════════════════════════════════════

    - 首次运行会下载AI模型（约140MB）
    - 推荐使用Base或Small模型平衡速度和质量
    - 音频质量越好，对齐效果越准确
    - 建议单个音频不超过30分钟
